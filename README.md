# PhotosView
Ionic2 九宫格图片展示和查看大图component
===
* 功能如下:
    * 1.九宫格图片展示
    * 2.点击查看大图
    * 3.分享至微信、qq

* 依赖插件:
    * $ ionic plugin add cordova-plugin-x-socialsharing
